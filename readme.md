## Información general
Este proyecto es un gestor de aulas desarrollado por alumnos de la Facultad de Informática de la UNLP, para la Facultad de Medicina. 
	
## Tecnologías
El proyecto fué creado con:
* Python 3
* Mysql 8
* Django 4
* Bootstrap 4
* JQuery
